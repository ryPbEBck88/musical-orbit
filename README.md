## Настройка формы заявки

1. Скопируйте `config.example.js` как `config.js`:
   ```bash
   cp config.example.js config.js
   ```

2. Откройте `config.js` и укажите один из вариантов:
   - **Google Sheets**: Следуйте инструкции в `GOOGLE_SHEETS_SETUP.md`
   - **FormSubmit**: Просто укажите ваш email в `FORMSUBMIT_EMAIL`

3. `config.js` уже добавлен в `.gitignore` и не будет закоммичен в репозиторий
