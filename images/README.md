# Изображения для сайта

Поместите следующие изображения в эту папку:

## Необходимые файлы:

1. **logo.png** - Логотип фестиваля "Музыкальная орбита"
   - Должен содержать: голубь с оливковой ветвью, текст "Музыкальная ОРБИТА" и "АССОЦИАЦИЯ МУЗЫКАЛЬНЫХ ТВОРЧЕСКИХ КОЛЛЕКТИВОВ"
   - Рекомендуемый размер: минимум 400x400px (для лучшего качества)
   - Формат: PNG с прозрачным фоном (предпочтительно) или JPG

2. **hero-space-1.jpg** - Космическое изображение для hero-секции
   - Должно содержать: музыкальные инструменты и планеты в космосе
   - Рекомендуемый размер: 1920x600px или больше (для адаптивности)
   - Формат: JPG или PNG

## Дополнительные изображения (опционально):

Если у вас есть другие космические изображения с музыкальными инструментами, их можно использовать для создания слайдера. Просто добавьте их как:
- hero-space-2.jpg
- hero-space-3.jpg
и т.д.
